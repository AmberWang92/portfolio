Furry Bloom Bounce

Unity Mobile Game Physics-Based Tilt Controls
View on GitHub

About the Project

A 2D mobile game prototype developed in Unity, where the player controls a small soot-like creature by tilting their phone. The goal is to navigate through obstacle-filled levels, activate switches to unlock paths, and reach the exit.

As the character moves, it leaves behind a trail of small flowers, turning traversal into a subtle, rewarding visual experience.

The project was successfully deployed and tested on iPhone, marking my first end-to-end mobile development workflow.


Challenges & Key Learnings

  • Gyro-Driven Controls – Tuned acceleration curves and damping so tilt feels responsive without being twitchy.
  • Switch & Gate Systems – Built modular triggers that unlock paths as the player solves light puzzles.
  • Device Testing – Iterated directly on iPhone hardware to validate tilt responsiveness and frame pacing.
  • Build Automation – Documented an end-to-end export pipeline (Unity → Xcode → TestFlight) for future prototypes.
