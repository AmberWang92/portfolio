Until the Fields: Farmageddon

Unity C# 3D Puzzle Platformer

About the Project

Until the Fields: Farmageddon is a 3D single-player puzzle platformer adventure where you swap between 3 animals: mouse, spider and cow — each with their own personality and tricks — Our heroes must cooperate to solve puzzles and escape the storm-wrecked barn!


The game is a student project developed by a team of 6 women over 3.5 months. I spend more than 500 hours on this project.

Gameplay state handler excerpt showing how we transition between animal forms with shared input routing.

// StateManager.cs
public void SwitchForm(AnimalForm nextForm)
{
    if (currentForm == nextForm) return;

    currentForm.OnExit();
    currentForm = forms[nextForm];
    currentForm.OnEnter();

    abilityBus.RouteTo(currentForm);
    uiSignals.Dispatch(FormChanged, currentForm.DisplayName);
}
